Установка VM VirtualBox

Скачать VM VirtualBox можно с официального сайта: virtualbox.org

Пример установки VirtualBox в Windows:

Выберите VirtualBox и скачайте установщик.

Запустите файл и нажмите Next.

Выберите путь установки → Next.

Выберите нужные опции → Next.

Подтвердите отключение сети → Yes.

Нажмите Install.

Дождитесь завершения → Finish.

Загрузка ISO Linux

Скачать ISO можно:
  • Ubuntu 16.04
  • CentOS 7

Создание виртуальной машины

Нажмите New.

Введите имя и выберите тип ОС.

Выделите RAM.

Создайте виртуальный диск.

Выберите размер и ISO файл → Start.

Установка Ubuntu Server

Выберите язык → English.

Начните установку → Enter.

Введите имя сервера.

Создайте пользователя.

Введите пароль.

Пароль не отображается при вводе — это нормально.

Завершите установку → Continue.

Готово! 🎉 Теперь можно переходить к лабораторным работам.
